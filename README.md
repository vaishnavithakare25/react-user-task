React User Management Application

Project Description:
This is a React + TypeScript application that manages user information using the DummyJSON API. The application allows users to view, search, add, edit, and delete user records. Local Storage is used to store newly added users.

Features:
User Login with validation using Zod
Dashboard page
User List page
Search users
Pagination
Sort users (A-Z and Z-A)
View user details
Add new user
Edit user information
Delete user
Display user role, status, and created date
Local Storage integration for added and updated users
Loading state while fetching data

Technologies Used:
React
TypeScript
React Router DOM
Axios
Tailwind CSS
Zustand
Zod

API Used:
DummyJSON API
https://dummyjson.com/users
